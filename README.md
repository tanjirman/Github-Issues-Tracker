1️⃣ What is the difference between var, let, and const?

Answer: Var- Var(old version) is a Function-scoped variable. It can be re-declared and updated. The main issue is that, it can be hoisted to the top of its scope.

let- let is a Block-scoped variable. It can be updated but not re-declared in the same scope. Hoisted but in a fixed zone.

const- const is a Block-scoped constant. It cannot be updated or re-declared. const must be initialized during declaration.

2️⃣ What is the spread operator (...)?

Answer: spread operator (...) used to expand iterables (arrays, objects, or strings) into individual elements. If any (arrays, objects, or strings) are made in later, it helps to copying, merging, or passing multiple values.

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer: map()-Iterates over an array and returns a new array with the results of applying a function to each element. Original array remains unchanged.

filter()-Iterates over an array and returns a new array containing only elements that satisfy a specific condition. It gives all the satisfied conditions output. Original array remains unchanged.

forEach()-Iterates over an array and performs a function on each element. Does not return a new array. Used for side effects.

4️⃣ What is an arrow function?

Answer: An arrow function is a concise way to write a JavaScript function using the () => {} syntax. It was introduced in ES6 to simplify function expressions, especially its optimized version of a regular function.

5️⃣ What are template literals?

Answer: Template Literals Strings enclosed in backticks (``) that allow dynamic behavior of variables and expressions using ${}. Supports multi-line strings without concatenation.
